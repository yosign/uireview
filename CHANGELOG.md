# 📋 更新日志

## v2.0.0 - 简化版本 (2025-12-16)

### ✨ 界面简化

#### 移除的功能
- ❌ API 配置界面输入（改为只支持环境变量）
- ❌ 风格选择下拉框（服务器端控制）
- ❌ 输出尺寸选择（服务器端控制）
- ❌ localStorage 配置保存

#### 保留的功能
- ✅ 图片上传和预览
- ✅ 动作描述输入
- ✅ 生成和下载功能
- ✅ 实时状态反馈

### 🎨 UI 改进

#### 标题和描述
- "步骤1: 生成精灵图" → "生成精灵图动画"
- 副标题更清晰地说明功能

#### 动作描述
- "提示词" → "动作描述"
- 占位符更新为具体的动作示例：
  - 跑步
  - 跳跃
  - 挥手
  - 攻击
  - 转身

#### 提示信息优化
- 更简洁的使用提示
- 未配置 API 时显示醒目提示
- 移除冗余信息

### 🔧 技术改进

#### 配置管理
- 强制使用环境变量配置
- 简化代码逻辑
- 移除不必要的 state 和 useEffect

#### API 请求
- 简化请求参数（移除 style 和 size）
- 保留核心参数：
  - `prompt` - 动作描述
  - `inputimage` - 上传的图片

### 📝 文档更新

#### 新增文档
- `QUICK_START.md` - 快速开始指南
- `CHANGELOG.md` - 本文件

#### 更新文档
- `env.example.txt` - 更清晰的配置说明
- 标注必填项和可选项

### 🎯 用户体验

#### 简化流程
1. 配置 API Key（一次性）
2. 上传图片
3. 描述动作（简洁直接）
4. 生成

#### 减少困惑
- 不再需要选择风格和尺寸
- 动作描述更明确
- 配置方式更统一

### ⚡ 性能优化

- 移除不必要的组件和状态
- 简化渲染逻辑
- 减少包大小

- Next.js 14 + Shadcn UI

- Dify API 集成
- CORS 问题修复（API 路由代理）
- GIF 编码库内嵌

### 配置方式
- 支持环境变量
- 支持界面输入
- 支持 localStorage 保存

### 完整功能

- 风格选择（4种）
- 尺寸选择（3种）
- 提示词输入
- 图片上传预览
- 实时状态反馈
